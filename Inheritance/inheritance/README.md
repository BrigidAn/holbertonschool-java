# Learning Objectives
By the end of this course, you are expected to be able to explain to anyone, without the help of Google:

> General
How to organize Java classes into packages
What is a superclass, base class, or parent class
What is a child class or subclass
How to inherit a class
What is the purpose of inheritance
What is super
What is @Override
What are abstract classes
What is instanceof
# Requirements
> Java Tasks
Allowed editors: Visual Studio Code, Eclipse, IntelliJ IDEA, NetBeans
Programs should not contain package, typically included by IDEs.
Only the source code file with the .java extension should be uploaded.
All files must be compiled on Ubuntu 20.04 LTS using javac
A README.md file, in the root of the project folder, is mandatory
